About

Hi there, my name is Ardian.

I’m a WordPress theme and plugin developer based in Jakarta, Indonesia, where I live with my wife and beloved daughter.

This website is a personal space where I share my web development journey. Here, you’ll find my latest projects, code snippets, tutorials, and insights — with a strong focus on WordPress plugin and block development.

I’m especially passionate about extending WordPress using the Gutenberg block editor. I believe the advent of blocks has made it significantly easier for developers to build custom, dynamic experiences in WordPress — allowing for more flexibility and a better user experience.

If you’re interested in the WordPress plugins I’m working on, check out the Projects page. One of my current works is a custom block called Larris GitHub Showcase, designed to help WordPress users present GitHub content more effectively. It’s part of my broader mission to create tools that meet real user needs and make working with WordPress even more enjoyable.

Thanks for stopping by!
